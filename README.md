# Bear Beer Catalog
- Projeto de um catálogo de cervejas

## Técnologias Utilizadas
- ReactJS
- Emotion/css - styled
- Material UI Icons
- Redux
- Redux Thunk
- Axios
- React Router Dom
- PunkApi
- Webpack

### Run

`git clone https://github.com/gabrielcarlossl/beer-catalog.git`

`npm install` 

`npm run dev`
